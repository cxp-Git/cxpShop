# cxpShop
仿天猫 电商网站

后台使用了Springboot、mybatis ，前端使用了vue、BootStrap，前后端分离通过json进行交互，商品列表部分进行分页，可视化的展示商品信息。

这是首页 搜索框可以根据关键词从数据库模糊查询出对应商品，中间的图片是轮播的
